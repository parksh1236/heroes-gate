# 영웅문: 화산논검 (Heroes Gate)

무협 로그라이크 덱빌딩 웹게임 — 진우/선우가 되어 화산논검 천하제일을 노려라!

## ▶️ 게임 실행 (클릭)

### 👉 **[https://parksh1236.github.io/heroes-gate/](https://parksh1236.github.io/heroes-gate/)**

브라우저(PC·모바일)에서 위 주소를 열면 바로 플레이할 수 있습니다. 설치 불필요.

## 게임 소개

- **전투**: 슬레이 더 스파이어 방식 — 매 턴 기력 3, 무공 카드 사용, 적 의도 읽고 공방 선택
- **연계**: 같은 문파(검·장·권·기) 카드를 연속 사용하면 크리티컬 확률 상승 + 3연속마다 연계기 발동
- **성장**: 3막 분기 지도, 무공 카드 20종, 비급 27종, 카드 연마/제거
- **스토리**: 김용 《영웅문》 세계관 — 매초풍, 구천인을 넘어 서독 구양봉과 최후의 비무

단일 HTML 파일(바닐라 JS), SVG 아트, Web Audio 합성 효과음.
